# Research Agent Frontend

React + Vite + Mantine frontend for the Research Paper Analyzer.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

The app will be available at `http://localhost:5173`

## Build for Production

```bash
npm run build
```

## Technologies

- React 18
- TypeScript
- Vite
- Mantine UI
- Axios
- react-markdown
